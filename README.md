# 🏭 Mangalam Pipes Website

A professional and responsive business website built for **Mangalam Pipes Pvt Ltd**, a leading manufacturer and supplier of HDPE pipes and fittings in India.

🔗 **Live Demo:** https://mangalamhdpepipess.netlify.app/

---

## 📌 About the Project

This website is designed to present **Mangalam Pipes’ products, services, and company profile** in a modern and user-friendly interface.

Mangalam Pipes is a well-established company specializing in:

* Manufacturing HDPE Pipes & Coils
* Pipe Fittings and Accessories
* Welding & Jointing Services ([mangalampipes.net][1])

The goal of this project is to create a **clean digital presence** for industrial clients, contractors, and businesses.

---

## ✨ Features

* 🌐 Fully responsive design (mobile + desktop)
* 🏢 Professional business layout
* 📦 Product showcase (HDPE Pipes, Fittings, Coils)
* 🛠️ Services section (Welding & Jointing)
* 📞 Contact information section
* ⚡ Fast loading & optimized UI
* 🎯 Clean navigation and UX
